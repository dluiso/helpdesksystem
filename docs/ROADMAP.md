# Roadmap

## Milestone 1

- Monorepo, Docker Compose, PostgreSQL, Redis
- NestJS API and Next.js dashboard shell
- Prisma schema and seed data
- Health check
- Auth skeleton with HttpOnly sessions
- Roles, groups, permissions
- Clients, domains, contacts
- Tickets, messages, attachments
- File storage abstraction
- Mail provider interfaces
- Rich text editor placeholder
- AI assistant mock provider
- Remote access abstraction

## Milestone 2

- Complete user, group, role, permission CRUD
- Full ticket CRUD and assignment workflow
- Multipart attachment upload
- TipTap or Lexical rich text editor integration
- CSRF protection
- Password-change workflow

## Milestone 3

- Microsoft Graph OAuth configuration
- Inbound mailbox sync
- Inbound attachment ingestion
- Outbound replies with threading and attachments
- Auto-reply history and throttling

## Later

- Reports and exports
- Knowledge base editor
- Device inventory refinement
- External remote access provider links
- Mobile app support
- Endpoint agent event channel
